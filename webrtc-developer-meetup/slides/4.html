Acquiring local media - OpenTok

var publisher = TB.initPublisher(apiKey, "localVideo", {
    publishAudio: true,
    publishVideo: true
});

(publishAudio and publishVideo are optional too)

Or with the AngularJS module (https://github.com/aullman/OpenTok-Angular).

<publisher apiKey="1127" props="{'publishAudio':true, 'publishVideo':true}"></publisher>
