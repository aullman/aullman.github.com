OpenTok - Connections and Streams

  • Connection
    • properties
      • connectionId - Unique identifier for this connection
      • data - Some custom data that you can pass when you connect
      • creationTime - When this connection was created
  • Stream
    • properties
      • streamId - Unique identifier for this stream
      • connection - Reference to the connection this stream is coming from
      • creationTime - When this stream was created
      • name - Custom name for this stream
